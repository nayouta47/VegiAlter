# 밭빌드 (가제) — 구현 사양서

**버전**: 0.3
**장르**: 로그라이크 밭빌딩 / 싱글플레이

---

## 1. 컨셉

씨앗 덱에서 드로우한 카드를 격자에 배치하여 식물을 키우고, 사전 정의된 위협 시퀀스를 식물로 막아내며, 상점에서 덱과 밭을 성장시켜 최종 보스까지 도달하는 게임. 핵심 트레이드오프: 수익 추구 vs 위협 대비. 같은 자원(카드, 물)을 두 목적이 경쟁적으로 소모한다.

---

## 2. 용어 정의

액션: 카드 1장 사용.
턴: 5장 드로우 + 물 3 회복. 물을 소비하여 복수 액션 수행.
라운드: 위협 배치 → 복수 턴 진행 → 모든 위협 처리 시 종료.
상점: 라운드 사이에 진행. 3택1 + 골드 소비 구매.
런: 복수 라운드 + 상점 + 보스전으로 구성되는 1회 플레이.

---

## 3. 턴 구조

턴 시작: 덱에서 5장 드로우. 물 3 회복.

액션: 물을 소비하여 손패의 카드를 사용. 카드마다 코스트가 다름.
- 야채 카드: 밭의 빈칸에 배치 (심기).
- 도구 카드: 효과 발동 후 discard.
- 물이 부족하거나 배치할 칸이 없으면 더 이상 액션 불가.

손패 잔여 처리: 사용하지 않은 손패는 전부 discard.

턴 종료 처리 (순서 중요):
1. 성장: 밭 위 모든 식물에 성장 스택 +1. 단, 성장 정지 상태인 식물은 스택을 받지 않음.
2. 풀스택 발동: 성장 스택이 풀에 도달한 식물은 자기 포함 인접 4칸(상하좌우)의 식물 체력을 1 회복. 이후 해당 식물의 성장 스택 리셋.
3. 위협 타이머 -1: 모든 위협의 타이머를 1 감소.
4. 위협 발동: 타이머가 0이 된 위협의 시퀀스 처리 (후술).

---

## 4. 덱 시스템

### 4.1 시작 덱

콩 7장, 마늘 3장, 물뿌리개 1장, 레인스틱 1장. 총 12장.

### 4.2 덱 순환

드로우 → 사용 또는 discard → 덱 소진 시 discard 전체를 셔플하여 새 덱 구성.
죽은 야채 카드(후술)도 discard에 합류하여 셔플 대상에 포함.

### 4.3 라운드 간 리셋

라운드 종료 시: 죽은 야채의 더미 태그 해제 (정상 카드로 복귀). 임시 태그 카드 전부 소멸. 나머지 카드 전부 합쳐서 셔플하여 새 덱 구성.

---

## 5. 카드 정의

### 5.1 야채 카드

야채 카드를 사용하면 밭의 빈칸에 식물로 배치된다. 식물은 체력과 성장 스택을 가진다.

콩: 코스트 1물. 체력 1. 풀스택 3.
마늘: 코스트 1물. 체력 3. 풀스택 3.

공통 규칙:
- 모든 야채는 턴 종료 시 성장 스택 +1.
- 풀스택 도달 시 자기 포함 인접 4칸 식물 체력 1 회복 후 스택 리셋.
- 식물이 사망하면 밭에서 제거되고, 해당 카드는 덱으로 돌아감. 단, 이번 라운드 동안 더미 태그가 붙어 드로우 시 사용 불가 (손패에서 아무 효과 없는 카드로 취급).

### 5.2 도구 카드

물뿌리개: 코스트 2물. 밭의 칸 1개를 지정. 해당 칸 기준 십자 범위(자기 포함 상하좌우)의 모든 식물에 성장 스택 +1. 사용 후 discard.

레인스틱: 코스트 0물. 덱에 "물방울" 카드 2장을 넣음. 사용 후 discard.

물방울(생성 카드): 코스트 0물. 사용 시 물 +1 회복. 사용 후 소멸. 임시 태그(라운드 종료 시 소멸).

---

## 6. 격자 밭

### 6.1 초기 상태

3×3, 9칸. 라운드 시작 시 빈 상태.

### 6.2 확장

상점에서 골드를 지불하여 1칸 단위로 확장. 확장 방향 선택 가능.

### 6.3 칸 상태

각 칸은 다음 상태 중 하나: 빈칸 / 식물 배치됨 (체력, 성장 스택, 성장 정지 여부 보유).
위협은 칸에 부착되며 식물과 공존.

---

## 7. 위협 시스템

### 7.1 구조

라운드 시작 시 해당 라운드의 모든 위협이 밭의 랜덤 칸에 배치됨. 각 위협은 타이머와 시퀀스를 가짐. 한 라운드의 위협 구성(개수, 강도, 타이머)은 런 시작 시 사전 정의됨. 배치 칸만 랜덤.

### 7.2 타이머

매 턴 종료 시 모든 위협 타이머 -1. 타이머 0 도달 시 해당 위협의 시퀀스 발동.

가장 긴 타이머의 위협이 발동하면 라운드 종료.

### 7.3 시퀀스

위협 1개는 순서가 있는 시퀀스로 구성됨. 요소 타입: 체력, 시간, 돈.
예시: [체력, 체력, 시간, 시간, 돈] → 5단계 시퀀스.

앞에서부터 1개씩 순서대로 처리한다.

### 7.4 시퀀스 처리 규칙

해당 칸에 살아있는 식물이 있는 경우:
- 체력 요소: 식물에 1 데미지. 식물 체력 0 이하 시 사망.
- 시간 요소: 해당 식물이 1턴 성장 정지. (다음 턴 종료 시 성장 스택을 받지 못함.)
- 돈 요소: 골드 1 획득.

처리 중 식물이 사망하면, 남은 시퀀스 전부 농부에게 전이:
- 체력 요소: 농부 체력 -1.
- 시간 요소: 시간 토큰 -1.
- 돈 요소: 획득 불가.

해당 칸이 비어있는 경우 (식물 없음): 시퀀스 전체가 농부에게 적용. 위와 동일.

### 7.5 동시 발동

같은 타이머 값의 위협이 복수 존재하면 같은 턴에 전부 발동. 처리 순서는 칸 순서(좌상→우하 등) 고정.

---

## 8. 자원

### 8.1 물

매 턴 3 회복. 카드 사용 시 코스트만큼 소모. 턴 간 이월 여부: 미확정 (프로토타입에서는 이월 없음, 매 턴 3으로 리셋 권장).

### 8.2 농부 체력

시작값 5. 0 도달 시 런 패배. 상점에서 회복 가능.

### 8.3 시간 토큰

런 시작 시 초기값 부여 (미확정). 특정 위협의 시간 요소가 농부에게 전이될 때 소모. 0 도달 시 보스 즉시 출현.

### 8.4 골드

위협 시퀀스의 돈 요소를 식물이 막아냈을 때 획득. 상점에서 소비.

---

## 9. 상점

### 9.1 타이밍

라운드 종료 후, 다음 라운드 시작 전.

### 9.2 구매

3택1 무료 선택: 매 상점마다 3개 선택지 제시, 1개 선택. 선택지 풀은 런마다 랜덤 구성.
골드 소비 구매: 추가 카드, 밭 확장, 농부 체력 회복 등.

### 9.3 구매 카테고리

새 야채 카드: 덱에 추가. 고유 스탯 및 풀스택 효과 보유.
도구 카드: 덱에 추가.
밭 확장: 격자 +1칸 (방향 선택).
농부 체력 회복.

---

## 10. 런 구조

### 10.1 런 진행

런 시작 → 시작 덱 구성, 시간 토큰 부여 → [라운드 → 상점] 반복 → 시간 토큰 0 시 보스전.

### 10.2 위협 스케줄

런 시작 시 전체 라운드의 위협 구성이 사전 정의됨. 라운드가 진행될수록 위협 수 증가, 타이머 감소, 시퀀스 강화.

### 10.3 보스전

보스도 일반 위협과 동일한 시스템으로 동작. 더 긴 시퀀스, 더 많은 동시 위협, 더 짧은 타이머.

### 10.4 승리 조건

보스전의 모든 위협을 처리하면 런 클리어.

### 10.5 패배 조건

농부 체력 0.

---

## 11. 미확정 항목

시간 토큰 초기값.
물의 턴 간 이월 여부.
밭 확장 비용.
상점 골드 구매 품목별 가격.
위협 스케줄 구체 데이터 (라운드별 위협 수, 타이머, 시퀀스 구성).
보스전 위협 데이터.
추가 야채 카드 종류 및 스탯.
추가 도구 카드 종류 및 효과.
메타 진행 (런 간 영구 해금 요소 유무).
밭 확장 형상 규칙 (직사각형 유지 vs 비정형 허용).
성장 정지가 복수 턴 중첩되는지 여부.
동시 발동 위협 간 처리 순서 확정 방식.
