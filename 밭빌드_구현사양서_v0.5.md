# 밭빌드 (가제) — 구현 사양서

**버전**: 0.5
**장르**: 로그라이크 밭빌딩 / 싱글플레이

---

## 1. 컨셉

씨앗 덱에서 드로우한 카드로 소수의 식물을 격자에 배치하고, 물주기로 성장을 관리하고, 위협이 공개되면 옮겨심기로 대응하며, 상점에서 덱과 밭을 성장시켜 최종 보스까지 도달하는 게임. 핵심 판타지: 물 새는 걸 허겁지겁 막는 느낌. 소수의 식물을 옮겨가며 위협에 대응하는 대응형 게임.

---

## 2. 용어 정의

액션: 카드 1장 사용.
턴: 5장 드로우 + 물 3 회복. 물을 소비하여 복수 액션 수행.
라운드: 위협 배치 → 복수 턴 진행 → 모든 위협 처리 시 종료.
상점: 라운드 사이에 진행. 3택1 + 골드 소비 구매.
런: 복수 라운드 + 상점 + 보스전으로 구성되는 1회 플레이.

---

## 3. 턴 구조

턴 시작: 덱에서 5장 드로우. 물 3 회복.

액션: 물을 소비하여 손패의 카드를 사용. 카드마다 코스트가 다름.
- 야채 카드: 밭의 빈칸에 배치 (심기).
- 물주기 카드: 대상 식물의 성장 스택 +1.
- 옮겨심기 카드: 밭 위의 식물을 다른 빈칸으로 이동.
- 도구 카드: 효과 발동 후 discard.
- 물이 부족하거나 유효 대상이 없으면 해당 카드 사용 불가.

손패 잔여 처리: 사용하지 않은 손패는 전부 discard.

턴 종료 처리 (순서 중요):
1. 성장: 밭 위 모든 식물에 성장 스택 +1. 단, 성장 정지 상태인 식물은 스택을 받지 않음.
2. 풀스택 발동: 성장 스택이 풀에 도달한 식물은 자기 포함 인접 4칸(상하좌우)의 식물에 방어 +1 부여. 이후 해당 식물의 성장 스택 리셋.
3. 위협 타이머 -1: 모든 위협의 타이머를 1 감소.
4. 위협 공개: 타이머가 0이 된 위협의 허들값과 시퀀스를 공개.
5. 위협 발동: 공개된 위협에 대해 허들 판정 및 시퀀스 처리.

---

## 4. 덱 시스템

### 4.1 시작 덱

콩 2장, 마늘 1장, 물주기 4장, 옮겨심기 5장. 총 12장.

### 4.2 카드 카테고리

야채 카드: 밭에 식물을 배치. 덱 내 비율이 낮음. 한 장의 가치가 높다.
물주기 카드: 기존 식물의 성장을 가속. 리듬 동기화의 핵심 수단.
옮겨심기 카드: 기존 식물의 위치를 변경. 위협 대응의 핵심 수단.
도구 카드: 상점에서 획득. 특수 효과.

### 4.3 덱 순환

드로우 → 사용 또는 discard → 덱 소진 시 discard 전체를 셔플하여 새 덱 구성. 죽은 야채 카드(후술)도 discard에 합류하여 셔플 대상에 포함.

### 4.4 라운드 간 리셋

라운드 종료 시: 밭 전체 초기화 (식물 전부 제거). 죽은 야채의 더미 태그 해제 (정상 카드로 복귀). 임시 태그 카드 전부 소멸. 나머지 카드 전부 합쳐서 셔플하여 새 덱 구성.

### 4.5 덱빌딩 긴장

덱 압축: 불필요 카드 제거 → 핵심 카드 드로우 확률 상승. 그러나 야채 부족 시 밭에 구멍 발생.
덱 확장: 야채 추가 → 밭 커버리지 확보. 그러나 덱이 두꺼워져 물주기/옮기기 드로우 확률 하락.
두 방향 모두 유효하도록 밸런싱 필요.

---

## 5. 카드 정의

### 5.1 야채 카드

야채 카드를 사용하면 밭의 빈칸에 식물로 배치된다. 식물은 체력, 방어, 성장 스택을 가진다. 방어는 0에서 시작한다.

콩: 코스트 1물. 체력 1. 풀스택 3.
마늘: 코스트 1물. 체력 3. 풀스택 3.

공통 규칙:
- 모든 야채는 턴 종료 시 성장 스택 +1.
- 풀스택 도달 시 자기 포함 인접 4칸 식물에 방어 +1 부여 후 스택 리셋.
- 식물이 사망하면 밭에서 제거되고, 해당 카드는 덱으로 돌아감. 단, 이번 라운드 동안 더미 태그가 붙어 드로우 시 사용 불가 (손패에서 아무 효과 없는 카드로 취급).

### 5.2 물주기 카드

물주기: 코스트 1물. 밭 위의 식물 1개를 지정. 해당 식물의 성장 스택 +1. 사용 후 discard.

### 5.3 옮겨심기 카드

옮겨심기: 코스트 0물. 밭 위의 식물 1개를 지정하여 다른 빈칸으로 이동. 식물의 체력, 방어, 성장 스택 모두 유지. 사용 후 discard.

### 5.4 도구 카드

시작 덱에는 포함되지 않음. 상점에서 획득.

물뿌리개 (상점 획득 예시): 코스트 2물. 밭의 칸 1개를 지정. 해당 칸 기준 십자 범위(자기 포함 상하좌우)의 모든 식물에 성장 스택 +1. 사용 후 discard.

레인스틱 (상점 획득 예시): 코스트 0물. 덱에 "물방울" 카드 2장을 넣음. 사용 후 discard.

물방울 (생성 카드): 코스트 0물. 사용 시 물 +1 회복. 사용 후 소멸. 임시 태그 (라운드 종료 시 소멸).

---

## 6. 격자 밭

### 6.1 초기 상태

시작 크기 미확정 (2×2 권장). 라운드 시작 시 빈 상태.

### 6.2 확장

라운드 종료 후 밭이 확장됨. 확장 속도는 빠름 — 라운드가 진행될수록 커버해야 할 칸이 야채 카드 수보다 빠르게 늘어남. 이것이 상점에서 야채 카드를 추가해야 하는 압력을 만든다.

### 6.3 칸 상태

각 칸은 다음 상태 중 하나: 빈칸 / 식물 배치됨.
식물 배치됨 상태의 속성: 체력 (양의 정수), 방어 (0 이상의 정수), 성장 스택 (0 이상), 성장 정지 잔여 턴 (0 이상).
위협은 칸에 부착되며 식물과 공존.

---

## 7. 위협 시스템

### 7.1 구조

라운드 시작 시 해당 라운드의 모든 위협이 밭의 랜덤 칸에 배치됨. 위협의 타이머와 배치 칸은 공개. 허들값과 시퀀스는 비공개 — 위협 발동 시점에 공개됨.

한 라운드의 위협 구성 (개수, 허들값, 시퀀스, 타이머)은 런 시작 시 사전 정의됨. 배치 칸만 랜덤.

### 7.2 부분 공개 규칙

라운드 시작 시 공개: 위협의 수, 각 위협의 칸 위치, 각 위협의 타이머.
발동 시 공개: 해당 위협의 허들값, 시퀀스 구성.

플레이어는 "어디에서 언제 터지는지"는 알지만, "얼마나 강한지"는 모른다. 이 불완전 정보가 사전 배치를 베팅으로 만들고, 공개 후 옮겨심기로 대응하는 흐름을 만든다.

### 7.3 타이머

매 턴 종료 시 모든 위협 타이머 -1. 타이머 0 도달 시 허들값과 시퀀스 공개 후 허들 판정 진행.

가장 긴 타이머의 위협이 처리되면 라운드 종료.

### 7.4 위협 데이터 구조

위협 1개는 다음으로 구성됨:
- 타이머: 양의 정수. 발동까지 남은 턴 수.
- 허들값: 양의 정수. 무효화에 필요한 (체력 + 방어) 임계치.
- 시퀀스: 순서 있는 요소 배열. 요소 타입은 체력, 시간, 돈.

예시: 타이머 3, 허들 3, 시퀀스 [체력, 체력, 시간, 시간, 시간, 돈].

### 7.5 허들 판정

위협 발동 시 해당 칸을 검사한다.

해당 칸에 살아있는 식물이 있는 경우:
- (식물의 체력 + 방어) >= 허들값 → 허들 성공. 시퀀스 내 체력/시간 요소 전부 무효화. 돈 요소는 정상 획득. 방어 소멸 (0으로 리셋).
- (식물의 체력 + 방어) < 허들값 → 허들 실패. 시퀀스 실행. 방어 소멸 (0으로 리셋).

해당 칸이 비어있는 경우: 허들 자동 실패. 시퀀스 전체가 농부에게 적용.

### 7.6 허들 실패 시 시퀀스 실행

시퀀스를 앞에서부터 1개씩 순서대로 처리한다.

식물이 살아있는 동안:
- 체력 요소: 식물에 1 데미지. 식물 체력 0 이하 시 사망.
- 시간 요소: 해당 식물에 성장 정지 1턴 부여.
- 돈 요소: 골드 1 획득.

시퀀스 처리 중 식물이 사망하면, 남은 시퀀스 전부 농부에게 전이:
- 체력 요소: 농부 체력 -1.
- 시간 요소: 시간 토큰 -1.
- 돈 요소: 획득 불가.

해당 칸이 비어있는 경우: 시퀀스 전체를 농부에게 적용. 위와 동일.

### 7.7 동시 발동

같은 타이머 값의 위협이 복수 존재하면 같은 턴에 전부 발동. 처리 순서는 칸 순서 (좌상→우하 등) 고정.

## 8. 자원

### 8.1 물

매 턴 3 회복. 카드 사용 시 코스트만큼 소모. 턴 간 이월 없음 (매 턴 3으로 리셋).

### 8.2 농부 체력

시작값 5. 0 도달 시 런 패배. 상점에서 회복 가능.

### 8.3 시간 토큰

런 시작 시 초기값 부여 (미확정). 위협 시퀀스의 시간 요소가 농부에게 전이될 때 소모. 0 도달 시 보스 즉시 출현.

### 8.4 골드

허들 성공 시 시퀀스 내 돈 요소 전부 획득. 허들 실패 시 식물이 살아있는 동안의 돈 요소만 획득 (식물 사망 후 돈 요소는 획득 불가). 상점에서 소비.

---

## 9. 상점

### 9.1 타이밍

라운드 종료 후, 다음 라운드 시작 전.

### 9.2 구매

3택1 무료 선택: 매 상점마다 3개 선택지 제시, 1개 선택. 선택지 풀은 런마다 랜덤 구성.
골드 소비 구매: 추가 카드, 농부 체력 회복 등.

### 9.3 구매 카테고리

새 야채 카드: 덱에 추가. 고유 스탯 및 풀스택 효과 보유. 밭 커버리지 확보.
도구 카드: 덱에 추가. 특수 효과.
물주기/옮기기 추가: 덱의 유연성 강화. 단 야채 드로우 확률 하락.
농부 체력 회복.
덱 압축 (카드 제거).

---

## 10. 런 구조

### 10.1 런 진행

런 시작 → 시작 덱 구성, 시간 토큰 부여 → [라운드 → 밭 확장 → 상점] 반복 → 시간 토큰 0 시 보스전.

### 10.2 위협 스케줄

런 시작 시 전체 라운드의 위협 구성이 사전 정의됨. 라운드가 진행될수록 위협 수 증가, 타이머 감소, 허들값 상승, 시퀀스 강화.

### 10.3 보스전

보스도 일반 위협과 동일한 시스템으로 동작. 더 높은 허들값, 더 긴 시퀀스, 더 많은 동시 위협, 더 짧은 타이머.

### 10.4 승리 조건

보스전의 모든 위협을 처리하면 런 클리어.

### 10.5 패배 조건

농부 체력 0.

## 11. 핵심 다이나믹

### 11.1 리듬 동기화

성장 리듬: 모든 식물이 매 턴 성장 +1. 물주기 카드로 가속 가능. 풀스택(3) 도달 시 인접에 방어 부여. 물주기의 사용 타이밍이 방어 펄스의 타이밍을 결정한다.

위협 리듬: 위협마다 고유한 타이머. 타이머 도달 시 허들 판정. 타이머는 공개되어 있으므로 "언제 터지는지"는 알 수 있다.

플레이어의 과제: 물주기로 풀스택 타이밍을 조절하여, 위협 발동 전에 방어가 올라가도록 맞추는 것.

### 11.2 불완전 정보와 대응

위협의 강도(허들값, 시퀀스)가 발동 시까지 비공개이므로, 사전 배치는 베팅이다. "이 칸에 마늘을 놓으면 웬만한 건 막겠지" vs "콩으로 충분할까?" 공개 후 판단이 틀렸으면 옮겨심기로 대응한다. 옮기기는 0코스트이지만 손패 1장을 소모하므로, 덱에 옮기기가 많을수록 유연하지만 다른 카드가 적어진다.

### 11.3 커버리지 압력

밭이 라운드마다 확장되므로, 야채 카드 수가 밭 크기를 따라가지 못하면 빈칸이 생기고 농부가 직접 맞는다. 상점에서 야채를 추가하면 커버리지는 확보되지만 덱이 두꺼워져 물주기/옮기기 드로우 확률이 하락한다. 덱 압축만 하면 효율은 좋지만 밭을 다 커버할 수 없다. 이 긴장이 덱빌딩의 핵심 딜레마이다.

---

## 12. 미확정 항목

시작 밭 크기 (2×2 권장).
밭 확장 규칙 (라운드당 확장량, 형상, 자동 vs 선택).
시간 토큰 초기값.
위협 스케줄 구체 데이터.
보스전 위협 데이터.
추가 야채 카드 종류 및 스탯.
추가 도구 카드 종류 및 효과.
